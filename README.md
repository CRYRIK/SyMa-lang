## 1. –í–≤–µ–¥–µ–Ω–∏–µ

**SyMa** (Symbolic Math / Stack Math) ‚Äî —ç—Ç–æ –ø—Ä–æ—Å—Ç–æ–π –∏–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∏—Ä—É–µ–º—ã–π —è–∑—ã–∫ –ø—Ä–æ–≥—Ä–∞–º–º–∏—Ä–æ–≤–∞–Ω–∏—è, –æ—Å–Ω–æ–≤–∞–Ω–Ω—ã–π –Ω–∞ —Å—Ç–µ–∫–µ –∏ –∏—Å–ø–æ–ª—å–∑—É—é—â–∏–π –û–±—Ä–∞—Ç–Ω—É—é –ü–æ–ª—å—Å–∫—É—é –ó–∞–ø–∏—Å—å (RPN). –Ø–∑—ã–∫ —Ä–∞–∑—Ä–∞–±–æ—Ç–∞–Ω —Å –∞–∫—Ü–µ–Ω—Ç–æ–º –Ω–∞ –º–∞—Ç–µ–º–∞—Ç–∏—á–µ—Å–∫–∏–µ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è, —Ä–∞–±–æ—Ç—É —Å –º–∞—Å—Å–∏–≤–∞–º–∏ (—Å –∏—Å–ø–æ–ª—å–∑–æ–≤–∞–Ω–∏–µ–º NumPy) –∏ –ø—Ä–µ–¥–ª–∞–≥–∞–µ—Ç –ª–∞–∫–æ–Ω–∏—á–Ω—ã–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å —Å —É–Ω–∏–∫–∞–ª—å–Ω—ã–º–∏ Unicode-—Å–∏–º–≤–æ–ª–∞–º–∏ –¥–ª—è –æ–ø–µ—Ä–∞—Ü–∏–π, –∞ —Ç–∞–∫–∂–µ —Ç–µ–∫—Å—Ç–æ–≤—ã–µ –∞–ª–∏–∞—Å—ã –∏ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å —Å–æ–∑–¥–∞–Ω–∏—è –ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏—Ö —Å–ª–æ–≤ (—Ñ—É–Ω–∫—Ü–∏–π) –¥–ª—è –ª—É—á—à–µ–π —á–∏—Ç–∞–µ–º–æ—Å—Ç–∏ –∏ —Å—Ç—Ä—É–∫—Ç—É—Ä–∏—Ä–æ–≤–∞–Ω–∏—è –∫–æ–¥–∞.

**–ö–ª—é—á–µ–≤—ã–µ –æ—Å–æ–±–µ–Ω–Ω–æ—Å—Ç–∏:**

*   **–°—Ç–µ–∫–æ–≤–∞—è –∞—Ä—Ö–∏—Ç–µ–∫—Ç—É—Ä–∞:** –í—Å–µ –¥–∞–Ω–Ω—ã–µ –∏ —Ä–µ–∑—É–ª—å—Ç–∞—Ç—ã –æ–ø–µ—Ä–∞—Ü–∏–π —Ö—Ä–∞–Ω—è—Ç—Å—è –∏ –ø–µ—Ä–µ–¥–∞—é—Ç—Å—è —á–µ—Ä–µ–∑ —Å—Ç–µ–∫.
*   **RPN (–û–±—Ä–∞—Ç–Ω–∞—è –ü–æ–ª—å—Å–∫–∞—è –ó–∞–ø–∏—Å—å):** –û–ø–µ—Ä–∞—Ç–æ—Ä—ã —Å–ª–µ–¥—É—é—Ç –∑–∞ –æ–ø–µ—Ä–∞–Ω–¥–∞–º–∏ (`5 3 +` –≤–º–µ—Å—Ç–æ `5 + 3`).
*   **–ì–∏–±–∫–∏–π —Å–∏–Ω—Ç–∞–∫—Å–∏—Å:** –ü–æ–¥–¥–µ—Ä–∂–∫–∞ —É–Ω–∏–∫–∞–ª—å–Ω—ã—Ö Unicode-—Å–∏–º–≤–æ–ª–æ–≤ –∏ –∏–Ω—Ç—É–∏—Ç–∏–≤–Ω–æ –ø–æ–Ω—è—Ç–Ω—ã—Ö —Ç–µ–∫—Å—Ç–æ–≤—ã—Ö –∞–ª–∏–∞—Å–æ–≤ –¥–ª—è –æ–ø–µ—Ä–∞—Ü–∏–π –∏ –∫–æ–Ω—Å—Ç–∞–Ω—Ç.
*   **–ò–Ω—Ç–µ–≥—Ä–∞—Ü–∏—è —Å NumPy:** –ú–æ—â–Ω—ã–µ –≤–æ–∑–º–æ–∂–Ω–æ—Å—Ç–∏ –¥–ª—è —Ä–∞–±–æ—Ç—ã —Å N-–º–µ—Ä–Ω—ã–º–∏ –º–∞—Å—Å–∏–≤–∞–º–∏ –∏ –Ω–∞—É—á–Ω—ã–º–∏ –≤—ã—á–∏—Å–ª–µ–Ω–∏—è–º–∏.
*   **–î–∏–Ω–∞–º–∏—á–µ—Å–∫–∞—è —Ç–∏–ø–∏–∑–∞—Ü–∏—è:** –¢–∏–ø –¥–∞–Ω–Ω—ã—Ö –æ–ø—Ä–µ–¥–µ–ª—è–µ—Ç—Å—è –∑–Ω–∞—á–µ–Ω–∏–µ–º, —Ö—Ä–∞–Ω—è—â–∏–º—Å—è –≤ –ø–µ—Ä–µ–º–µ–Ω–Ω–æ–π –∏–ª–∏ –Ω–∞ —Å—Ç–µ–∫–µ.
*   **–ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–µ —Å–ª–æ–≤–∞ (–§—É–Ω–∫—Ü–∏–∏):** –í–æ–∑–º–æ–∂–Ω–æ—Å—Ç—å –æ–ø—Ä–µ–¥–µ–ª–µ–Ω–∏—è –∏ –≤—ã–∑–æ–≤–∞ —Å–æ–±—Å—Ç–≤–µ–Ω–Ω—ã—Ö –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ—Å—Ç–µ–π –∫–æ–º–∞–Ω–¥.
*   **–ò–Ω—Ç–µ—Ä–∞–∫—Ç–∏–≤–Ω—ã–π REPL:** –£–¥–æ–±–Ω–∞—è —Å—Ä–µ–¥–∞ –¥–ª—è —ç–∫—Å–ø–µ—Ä–∏–º–µ–Ω—Ç–æ–≤ –∏ –±—ã—Å—Ç—Ä—ã—Ö –≤—ã—á–∏—Å–ª–µ–Ω–∏–π —Å –∏—Å—Ç–æ—Ä–∏–µ–π –∫–æ–º–∞–Ω–¥ –∏ –∞–≤—Ç–æ–∑–∞–º–µ–Ω–æ–π.

## 2. –ö–∞–∫ –†–∞–±–æ—Ç–∞–µ—Ç –Ø–∑—ã–∫

–ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ç–æ—Ä SyMa –ø–æ—Å–ª–µ–¥–æ–≤–∞—Ç–µ–ª—å–Ω–æ —á–∏—Ç–∞–µ—Ç –∏ –æ–±—Ä–∞–±–∞—Ç—ã–≤–∞–µ—Ç –≤—Ö–æ–¥–Ω–æ–π –∫–æ–¥ (–∏–∑ —Ñ–∞–π–ª–∞ –∏–ª–∏ REPL), –≤—ã–ø–æ–ª–Ω—è—è —Å–ª–µ–¥—É—é—â–∏–µ —à–∞–≥–∏ –≤ —Ü–∏–∫–ª–µ (`run`):

1.  **–ü—Ä–æ–ø—É—Å–∫:** –ü—Ä–æ–±–µ–ª—ã, —Å–∏–º–≤–æ–ª—ã –Ω–æ–≤–æ–π —Å—Ç—Ä–æ–∫–∏ –∏ –∫–æ–º–º–µ–Ω—Ç–∞—Ä–∏–∏ (`//...`) –∏–≥–Ω–æ—Ä–∏—Ä—É—é—Ç—Å—è.
2.  **–ü–∞—Ä—Å–∏–Ω–≥ –¢–æ–∫–µ–Ω–∞ (`_parse_token`):** –ò–Ω—Ç–µ—Ä–ø—Ä–µ—Ç–∞—Ç–æ—Ä —á–∏—Ç–∞–µ—Ç —Å–ª–µ–¥—É—é—â–∏–π –∑–Ω–∞—á–∞—â–∏–π —Ñ—Ä–∞–≥–º–µ–Ω—Ç –∫–æ–¥–∞ (—Ç–æ–∫–µ–Ω) —Å —Ç–µ–∫—É—â–µ–π –ø–æ–∑–∏—Ü–∏–∏. –û–Ω –ø—ã—Ç–∞–µ—Ç—Å—è —Ä–∞—Å–ø–æ–∑–Ω–∞—Ç—å:
    *   **–ß–∏—Å–ª–∞:** (`123`, `-5.0`, `1e-3`, `inf`) -> `NUMBER`
    *   **–°—Ç—Ä–æ–∫–∏:** (`"—Ç–µ–∫—Å—Ç"`) -> `STRING`
    *   **–ë–ª–æ–∫–∏ –ö–æ–¥–∞:** (`{...}`) -> `BLOCK`
    *   **–ú–∞—Å—Å–∏–≤—ã:** (`[...]`) -> `ARRAY` (–ø—Ä–µ–æ–±—Ä–∞–∑—É–µ—Ç—Å—è –≤ NumPy array)
    *   **–°–ª–æ–≤–∞/–ò–¥–µ–Ω—Ç–∏—Ñ–∏–∫–∞—Ç–æ—Ä—ã:** (`add`, `pi`, `myVar`, `square`)
        *   –ï—Å–ª–∏ —Å–ª–æ–≤–æ - –∏–∑–≤–µ—Å—Ç–Ω—ã–π –∞–ª–∏–∞—Å –æ–ø–µ—Ä–∞—Ü–∏–∏ (`add`, `sin`, `store`) -> `OPERATOR`
        *   –ï—Å–ª–∏ —Å–ª–æ–≤–æ - –∏–∑–≤–µ—Å—Ç–Ω–∞—è –ø–µ—Ä–µ–º–µ–Ω–Ω–∞—è/–∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ (`pi`, `myVar`)
            *   –ï—Å–ª–∏ –∑–Ω–∞—á–µ–Ω–∏–µ - –±–ª–æ–∫ –∫–æ–¥–∞ -> `USER_WORD`
            *   –ò–Ω–∞—á–µ -> `VARIABLE_WORD`
        *   –ï—Å–ª–∏ –∑–∞ —Å–ª–æ–≤–æ–º —Å–ª–µ–¥—É–µ—Ç `‚Üí` –∏–ª–∏ `=` (–∏ —Å—Ç–µ–∫ –Ω–µ –ø—É—Å—Ç) -> `IDENTIFIER_ASSIGN` (–∏–ª–∏ `ASSIGN_VIA_EQ`)
        *   –ò–Ω–∞—á–µ (–Ω–µ–∏–∑–≤–µ—Å—Ç–Ω–æ–µ —Å–ª–æ–≤–æ) -> `NameError`
    *   **–û–¥–Ω–æ—Å–∏–º–≤–æ–ª—å–Ω—ã–µ –û–ø–µ—Ä–∞—Ç–æ—Ä—ã/–ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ:** (`+`, `‚àö`, `œÄ`, `‚Üí`, `=`)
        *   –ï—Å–ª–∏ —Å–∏–º–≤–æ–ª - –æ–ø–µ—Ä–∞—Ü–∏—è (`+`, `‚àö`, `=`) -> `OPERATOR`
        *   –ï—Å–ª–∏ —Å–∏–º–≤–æ–ª - –∫–æ–Ω—Å—Ç–∞–Ω—Ç–∞ (`œÄ`, `ùëí`) -> `VARIABLE_SYM`
    *   –ï—Å–ª–∏ —Å–∏–º–≤–æ–ª –Ω–µ —Ä–∞—Å–ø–æ–∑–Ω–∞–Ω -> `SyntaxError`
3.  **–û–±—Ä–∞–±–æ—Ç–∫–∞ –¢–æ–∫–µ–Ω–∞ (`run`):**
    *   **–ó–Ω–∞—á–µ–Ω–∏—è** (`NUMBER`, `STRING`, `BLOCK`, `ARRAY`, `VARIABLE_SYM`, `VARIABLE_WORD`): –ü–æ–º–µ—â–∞—é—Ç—Å—è –Ω–∞ —Å—Ç–µ–∫ (`push`).
    *   **–ò–º–µ–Ω–∞ –¥–ª—è –ø—Ä–∏—Å–≤–∞–∏–≤–∞–Ω–∏—è** (`IDENTIFIER_ASSIGN`, `ASSIGN_VIA_EQ`): –ü–æ–º–µ—â–∞—é—Ç—Å—è –Ω–∞ —Å—Ç–µ–∫ (–æ–∂–∏–¥–∞—è –≤—ã–∑–æ–≤–∞ `store`/`‚Üí`/`=`).
    *   **–û–ø–µ—Ä–∞—Ç–æ—Ä—ã** (`OPERATOR`): –í—ã–∑—ã–≤–∞–µ—Ç—Å—è —Å–æ–æ—Ç–≤–µ—Ç—Å—Ç–≤—É—é—â–∏–π –º–µ—Ç–æ–¥ `op_*` (–Ω–∞–ø—Ä–∏–º–µ—Ä, `op_add`, `op_store`). –û–ø–µ—Ä–∞—Ü–∏—è –±–µ—Ä–µ—Ç –∞—Ä–≥—É–º–µ–Ω—Ç—ã —Å–æ —Å—Ç–µ–∫–∞ (`pop`) –∏ –ø–æ–º–µ—â–∞–µ—Ç —Ä–µ–∑—É–ª—å—Ç–∞—Ç (`push`).
    *   **–ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–µ –°–ª–æ–≤–∞** (`USER_WORD`): –í—ã–∑—ã–≤–∞–µ—Ç—Å—è –º–µ—Ç–æ–¥ `_execute_block`, –∫–æ—Ç–æ—Ä—ã–π —Ä–µ–∫—É—Ä—Å–∏–≤–Ω–æ –∑–∞–ø—É—Å–∫–∞–µ—Ç `run` –¥–ª—è –∫–æ–¥–∞ –≤–Ω—É—Ç—Ä–∏ –±–ª–æ–∫–∞, –∏—Å–ø–æ–ª—å–∑—É—è —Ç–æ—Ç –∂–µ —Å—Ç–µ–∫ –∏ –ø–µ—Ä–µ–º–µ–Ω–Ω—ã–µ.
4.  **–ü—Ä–æ–¥–≤–∏–∂–µ–Ω–∏–µ:** –£–∫–∞–∑–∞—Ç–µ–ª—å —á—Ç–µ–Ω–∏—è –ø–æ—Ç–æ–∫–∞ —É—Å—Ç–∞–Ω–∞–≤–ª–∏–≤–∞–µ—Ç—Å—è –Ω–∞ –ø–æ–∑–∏—Ü–∏—é —Å—Ä–∞–∑—É –ø–æ—Å–ª–µ –æ–±—Ä–∞–±–æ—Ç–∞–Ω–Ω–æ–≥–æ —Ç–æ–∫–µ–Ω–∞.
5.  **–ü–æ–≤—Ç–æ—Ä–µ–Ω–∏–µ:** –¶–∏–∫–ª –ø—Ä–æ–¥–æ–ª–∂–∞–µ—Ç—Å—è –¥–æ –∫–æ–Ω—Ü–∞ –∫–æ–¥–∞ –∏–ª–∏ –≤–æ–∑–Ω–∏–∫–Ω–æ–≤–µ–Ω–∏—è –æ—à–∏–±–∫–∏.

## 3. –¢–∏–ø—ã –î–∞–Ω–Ω—ã—Ö –∏ –°–∏–Ω—Ç–∞–∫—Å–∏—Å

*   **–ß–∏—Å–ª–∞:** –¶–µ–ª—ã–µ (`int`), –í–µ—â–µ—Å—Ç–≤–µ–Ω–Ω—ã–µ (`float`), –ö–æ–º–ø–ª–µ–∫—Å–Ω—ã–µ (`complex`), –ë–µ—Å–∫–æ–Ω–µ—á–Ω–æ—Å—Ç—å (`inf`). –ó–∞–ø–∏—Å—ã–≤–∞—é—Ç—Å—è —Å—Ç–∞–Ω–¥–∞—Ä—Ç–Ω–æ: `123`, `-5`, `3.14`, `1.5e-3`.
*   **–õ–æ–≥–∏—á–µ—Å–∫–∏–µ:** `True` (`‚ä§`) –∏ `False` (`‚ä•`). –¢–∞–∫–∂–µ –º–æ–∂–Ω–æ –∏—Å–ø–æ–ª—å–∑–æ–≤–∞—Ç—å —Å–ª–æ–≤–∞ `True` –∏ `False`.
*   **–°—Ç—Ä–æ–∫–∏:** –í –¥–≤–æ–π–Ω—ã—Ö –∫–∞–≤—ã—á–∫–∞—Ö (`"`). –ü—Ä–∏–º–µ—Ä: `"–ü—Ä–∏–≤–µ—Ç"`, `"Result: "`.
*   **–ú–∞—Å—Å–∏–≤—ã NumPy:** –í –∫–≤–∞–¥—Ä–∞—Ç–Ω—ã—Ö —Å–∫–æ–±–∫–∞—Ö `[]`. –≠–ª–µ–º–µ–Ω—Ç—ã —Ä–∞–∑–¥–µ–ª—è—é—Ç—Å—è –∑–∞–ø—è—Ç—ã–º–∏ (`,`), —Å—Ç—Ä–æ–∫–∏ (–¥–ª—è 2D) ‚Äî —Ç–æ—á–∫–æ–π —Å –∑–∞–ø—è—Ç–æ–π (`;`). –ü—Ä–∏–º–µ—Ä: `[1, 2, 3]`, `[[1, 0]; [0, 1]]`. –°–æ–∑–¥–∞—é—Ç—Å—è —Å –ø–æ–º–æ—â—å—é `eval` (–±—É–¥—å—Ç–µ –æ—Å—Ç–æ—Ä–æ–∂–Ω—ã –ø—Ä–∏ –≤–≤–æ–¥–µ –∏–∑ –Ω–µ–Ω–∞–¥–µ–∂–Ω—ã—Ö –∏—Å—Ç–æ—á–Ω–∏–∫–æ–≤).
*   **–ë–ª–æ–∫–∏ –ö–æ–¥–∞:** –í —Ñ–∏–≥—É—Ä–Ω—ã—Ö —Å–∫–æ–±–∫–∞—Ö `{}`. –°–æ–¥–µ—Ä–∂–∞—Ç –≤–∞–ª–∏–¥–Ω—ã–π –∫–æ–¥ SyMa. –ü—Ä–∏–º–µ—Ä: `{ dup √ó }`. –ü—Ä–µ–¥—Å—Ç–∞–≤–ª—è—é—Ç —Å–æ–±–æ–π —Å—Ç—Ä–æ–∫–∏ –¥–æ –º–æ–º–µ–Ω—Ç–∞ –≤—ã–ø–æ–ª–Ω–µ–Ω–∏—è.
*   **–ü–µ—Ä–µ–º–µ–Ω–Ω—ã–µ:**
    *   **–í—Å—Ç—Ä–æ–µ–Ω–Ω—ã–µ –∫–æ–Ω—Å—Ç–∞–Ω—Ç—ã:** `œÄ`, `ùëí`, `ùëñ`, `‚àû`, `‚ä§`, `‚ä•` –∏ –∏—Ö —Ç–µ–∫—Å—Ç–æ–≤—ã–µ –∞–ª–∏–∞—Å—ã `pi`, `tau`, `e`, `j`, `inf`, `True`, `False`.
    *   **–ü–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–µ:** –ò–º–µ–Ω–∞ –Ω–∞—á–∏–Ω–∞—é—Ç—Å—è —Å –±—É–∫–≤—ã –∏–ª–∏ `_`, —Å–æ–¥–µ—Ä–∂–∞—Ç –±—É–∫–≤—ã, —Ü–∏—Ñ—Ä—ã, `_`. –°–æ–∑–¥–∞—é—Ç—Å—è —á–µ—Ä–µ–∑ `store`, `‚Üí`, `assign` –∏–ª–∏ `=`.
*   **–û–ø–µ—Ä–∞—Ü–∏–∏/–°–ª–æ–≤–∞:** –û–¥–∏–Ω–æ—á–Ω—ã–µ Unicode-—Å–∏–º–≤–æ–ª—ã –∏–ª–∏ —Ç–µ–∫—Å—Ç–æ–≤—ã–µ –∞–ª–∏–∞—Å—ã/–ø–æ–ª—å–∑–æ–≤–∞—Ç–µ–ª—å—Å–∫–∏–µ —Å–ª–æ–≤–∞.
